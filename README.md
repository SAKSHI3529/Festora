# 🎉 Festora – Cultural Event Management System

A full-stack web application designed to streamline the management of college cultural events including event creation, participant registration, scheduling, judging, and result declaration.

---

## 🚀 Features

- 🔐 Role-Based Access Control (RBAC)
  - Admin, Faculty Coordinator, Event Coordinator, Judges, Students

- 📝 Event Management
  - Create, update, and manage multiple events

- 👥 Participant Registration
  - Students can register for events easily

- 📊 Judging & Evaluation
  - Judges can assign scores and evaluate participants

- 🗓️ Scheduling System
  - Manage event timelines and coordination

- 📢 Result Declaration
  - Publish results dynamically

---

## 🛠️ Tech Stack

**Frontend**
- ReactJS
- HTML, CSS, JavaScript

**Backend**
- Python
- REST APIs

**Database**
- MongoDB

---

## 🧱 System Architecture

Festora follows a **3-tier architecture**:

1. **Presentation Layer** – React frontend  
2. **Application Layer** – Backend APIs (Python)  
3. **Data Layer** – MongoDB  

This ensures scalability, maintainability, and clean separation of concerns.

---

## 🔗 API Integration

- REST APIs handle communication between frontend and backend  
- Dynamic data flow for:
  - Event details  
  - User roles  
  - Registrations  
  - Results
